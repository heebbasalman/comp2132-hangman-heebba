# COMP2132 Hangman - Save the Buddy

A fun arcade-style Hangman game for the COMP2132 project.

## Features
- Random word and hint loaded from `data/words.json` using `fetch()`
- Letter buttons disable after being guessed
- Buddy character changes as mistakes increase
- Win/loss detection and play again option
- LocalStorage stats: wins, losses, streak, and best streak
- Hint power-up
- Shield power-up
- JavaScript animations: fade-in, screen shake, buddy jump, confetti
- 6+ image assets included
- CSS compiled from SASS
- SASS mixin included in `css/styles.scss`

## How to Run
Because this project uses `fetch()`, open it with Live Server or another local web server.

Example:
1. Open the folder in VS Code
2. Right-click `index.html`
3. Choose **Open with Live Server**
